Rules the build obeys
  • Money is ISO code + tabular mono. PHP 840, VND 378,000 — codes read truer than symbols across currencies, and columns align to the digit.
  • Sequence is real time. Rows are stamped with the clock — 08:32, 12:15 — never decorative 01 / 02 / 03. Order carries the day.
  • Continuity is the accounting. Unbroken tape = time accounted for. A break = money you forgot. Recovering it seals the break.
  • Two temperatures, load-bearing. Teal, amber and red are never decoration — each one means a specific state, always paired with a word.
  • Flat and printed. Perforations, dotted leaders, tabular rules, one stamp. No paper texture, no drop shadow, no faux fade.

# Embedded fonts

`src/fonts.css` inlines the following typefaces as base64 `woff2` data-URIs. All three
are licensed under the **SIL Open Font License, Version 1.1**
(<https://scripts.sil.org/OFL>). No modifications were made to the font files; only the
Latin subset is embedded.

| Family | Role in this design | Author(s) | Source |
| --- | --- | --- | --- |
| **Bricolage Grotesque** | Display — ritual headers only | Mathieu Triay | <https://github.com/ateliertriay/bricolage> |
| **Hanken Grotesk** | UI / body | Alfredo Marco Pradil | <https://github.com/marcologous/hanken-grotesk> |
| **Fragment Mono** | Figures / ledger | Wei Huang | <https://github.com/weiweihuanghuang/fragment-mono> |

Each family retains its own copyright and reserved font name under the OFL. See the
upstream repositories for the full license text and copyright notices. The OFL permits
embedding in documents; the fonts here are bundled solely to render this concept design
self-contained.
